> ## 该案例主要帮我同学们理解生命周期的使用以及react中的数据的流向

## 需求
```markdown
- 渲染一个列表 点击按钮添加
- 总数是奇数时每次点击额外添加两个 偶数是正常添加
- 总数大于20 且 小于 30页面不渲染
- 大于30页面正常渲染
- 每个块都要有一个序号，按顺序排列
- 页面数据
[
  {
    name: '千锋'
  }
]
```
