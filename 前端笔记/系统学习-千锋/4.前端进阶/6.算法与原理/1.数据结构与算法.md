#### 1. 集合（Set）

| 方法        | 作用                                                         |
| ----------- | ------------------------------------------------------------ |
| new Set([]) | 创建 Set ，接收一个数组                                      |
| [...set]    | 将 Set 转为数组                                              |
| add()       | 向集合中添加一个元素                                         |
| delete()    | 从集合中删除一个元素                                         |
| has()       | 查看集合中是否有该元素                                       |
| clear()     | 删除集合中的所有元素                                         |
| size()      | 返回集合中元素的数量                                         |
| values()    | 返回一个包含集合中所有元素的数组，但需要 `set.values().next()` 这样使用 |
|             |                                                              |
