#### 1. 组件基础

##### 1.1. 非单文件组件

```js
// 全局组件
Vue.component('comp-a',{
  template: `<div>全局组件-A组件 <comp-b></comp-b></div>`,
  data(){ return {} },
  methods: {},
  // 局部组件
  components: {
    'comp-b': {
      template: `<h2>局部组件-B组件</h2>`,
      data(){ return {} },
    }
  }
})
```

##### 1.2. 单文件组件

```vue
<template>
	<p>hello word!</p>
</template>
<script>
	export default {
    data() {
      return {}

  }
</script>
```
