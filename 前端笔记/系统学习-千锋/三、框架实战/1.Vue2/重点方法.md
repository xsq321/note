| 方法                       | 作用                                                         |
| -------------------------- | ------------------------------------------------------------ |
| Object.defineProperty()    | 为对象添加或修改属性，使之变为响应式的；Vue 通过该方法实现数据代理 |
| obj.hasOwnProperty('prop') | 判断对象本身是否含有某个属性，不包含原型链                   |
| this.$forceUpdate()        | 强制刷新                                                     |
| this.$nextTick(回调函数)   | 在下一次 DOM 更新结束后执行其指定的回调                      |
|                            |                                                              |
|                            |                                                              |
