#### 1. cookie

现已淘汰，使用 storage 代替，如 localeStorage , sessionStorage

Cookie 的使用：https://www.cnblogs.com/hhc16619804375/p/14906787.html

```js
document.cookie = "count=1;expires=xxx";  // 添加，expires 可用于设置过期时间
document.cookie = "count=2";  // 修改，只要键名相同就会自动找到原先的 cookie 进行修改
let cookies = document.cookie;  // 读取 cookie
```

虽然提供了 api 供前端使用 cookie，但实际开发中一般是由后端进行相应的设置，前端不用管 cookie 的使用
