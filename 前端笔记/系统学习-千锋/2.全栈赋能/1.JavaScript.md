#### 1. Object.defineProperty

对象自定义属性

```js
Object.defineProperty(obj, "a", {
    configurable: true, // 是否可被删除
    enumerable: true, // 是否可枚举
    writable: true, // 是否可被修改，以上三个默认都为 false
    value: "hello",  // 默认为 undefined
});
```

```js
Object.defineProperty(obj, "a", {
  get() {  // 访问时触发
    return "h";
  },
  set(val) {},  // 修改时触发
});
```

以上两者不能同时存在
