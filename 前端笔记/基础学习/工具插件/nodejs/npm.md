| 命令                    | 作用                         |
| ----------------------- | ---------------------------- |
| npm ls axios            | 查看本地已安装的 axios 版本  |
| npm ls axios -g         | 查看本地已全局安装的版本信息 |
| npm view axios versions | 查看所有版本信息             |
| npm view axios version  | 查看最新版本信息             |
| npm info axios          | 查看完整版本信息             |
