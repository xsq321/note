#### 2. 单文件组件

1.文件后缀是vue，2.命名规范：首字母大写及驼峰     3.下载插件：Vetur (用于提示和高亮等)   

4.注释用/* */     5.只支持三个标签：template, script, style   

##### 2.1. 简写

```javascript
<script>
    export default {		//省略了Vue.extend()
		name:'School',
        data(){},
        methods:{}
	}
```
